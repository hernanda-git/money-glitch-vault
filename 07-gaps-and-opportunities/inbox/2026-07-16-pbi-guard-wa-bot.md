# 2026-07-16-pbi-guard-wa-bot

**Source pain:** BPJS PBI Mendadak Nonaktif (demand-mining/bpjs-pbi-nonaktif-mendadak-warga-miskin.md)
**Cross-link:** converges with existing gap `2026-07-15-desil-dtks-checker-saas.md` (warga miskin terblokir status desil).

## The pitch
Warga miskin yang selama ini gratis berobat via BPJS PBI tiba-tiba dinonaktifkan massal (164rb di Sukabumi, 9.920 di Malang, skala nasional via SK Mensos No.3/HUK/2026). Mereka tidak tahu statusnya sampai sakit dan ditolak RS. Bangun bot WhatsApp "PBI Guard": cek status PBI via NIK, kalau nonaktif pandu reaktivasi (SKTM digital, arahkan Dinsos, template aduan DPRD). Versi B2G: dashboard kelurahan untuk deteksi dini warga PBI nonaktif sebelum butuh berobat. Pemda sudah siapkan ratusan miliar (Malang Rp 170 M) untuk selamatkan PBI, jadi ada anggaran untuk alat pencegah murah. Gabungkan dengan desil/DTKS checker jadi satu "status sosial guard" suite.

## Why now
Penonaktifan berlaku 1 Feb 2026, ramai sampai Komisi IX DPR panggil Menkes-BPJS. Pain masih panas di Juli 2026.
